## 🎯 Objetivos da Disciplina

Este repositório contém os scripts em **Python** desenvolvidos ao longo da disciplina. O foco cobre todas as etapas do processo de análise de dados:

- Utilização das principais bibliotecas: `NumPy`, `Pandas`, `Matplotlib`, `Seaborn` e `Streamlit`
- Desenvolvimento em Python com **VS Code**, ambientes virtuais, `uv`, Git e GitHub
- Visualizações para compreensão do comportamento dos dados e apresentação de resultados
- Interação com bancos de dados relacionais e consumo de arquivos `TXT`, `XLSX`, `CSV` e `JSON`
- Desenvolvimento de projeto aplicado extensionista de análise de dados para solução de problemas reais
